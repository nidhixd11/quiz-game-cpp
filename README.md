# C++ Quiz Game

This is a console-based quiz game built using C++. The program reads quiz questions from a text file, displays them to the user, checks the answers, and calculates the final score.

## Features

- Console-based quiz game
- Reads questions from a text file
- Takes user answers as input
- Checks correct answers
- Displays final score
- Uses file handling in C++

## Tech Stack

- C++
- File Handling
- Console I/O

## Files

- `quiz.cpp` - Main source code
- `quiz.txt` - Quiz questions file

## How to Run

Compile the program:

```bash
g++ quiz.cpp -o quiz.exe
